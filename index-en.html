📱 Ton site est-il responsive ? (adapté aux téléphones)

Un site "version téléphone" veut dire qu’il s’adapte à l’écran du visiteur (grand écran ou petit écran). On appelle ça un site responsive.

Voici comment savoir si ton site est responsive :

  1. Ouvre ton site dans un navigateur (par exemple Google Chrome).

  2. Rétrécis la fenêtre comme si c’était un écran de téléphone.

  3. Regarde : est-ce que le texte, les images et les menus s’adaptent bien ? Ou est-ce que tout devient trop petit ou mal placé ?

✨ Si ton site n’est pas encore responsive, voici ce qu’on peut faire :

  1. Ajouter cette ligne dans ton fichier HTML (dans <head>) :

    html
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    👉 Cette ligne dit au navigateur de s’adapter à la taille de l’écran.

  2. Utiliser du CSS responsive (pour que les blocs se placent bien sur téléphone)
    Par exemple :

    css
    body {
  font-family: Arial, sans-serif;
  padding: 10px;
  margin: 0;
}

.menu {
  display: flex;
  flex-wrap: wrap;
  gap: 10px;
}

@media (max-width: 600px) {
  .menu {
    flex-direction: column;
  }
}

  3. Tester sur téléphone réel ou dans les outils du navigateur :
    Sur Chrome, clique droit > Inspecter > en haut tu verras un bouton pour tester comme si c’était un téléphone.
